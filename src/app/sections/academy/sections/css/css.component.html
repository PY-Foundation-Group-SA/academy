CSS Lebenslauf
Verändere deinen Lebenslauf mit CSS. Verwende dazu alle Möglichkeiten, wie du CSS einbinden sowie HTML Elemente verändern kannst mindestens ein Mal. Gehe dabei wie folgt vor:
Verändere den Hintergrund, indem du zum Beispiel eine Farbe oder ein Bild festlegst
Die HTML Standard Font ist ziemlich langweilig, suche nach einer modernen Schrift und binde sie in deinen Lebenslauf ein. Verschiedene Fonts findest du unter anderem hier
Hebe die Überschrift noch mehr hervor, indem du ihr z.B. eine andere Farbe, Größe oder ähnliches gibst
Verändere dein Bild, indem du zum Beispiel die Größe oder Border anpasst, Margin oder Padding einfügst
Vereinheitliche die Farbe oder Größe von kursiv oder fett gedruckten Texten, indem du dessen Selektoren gruppierst und so die entsprechenden Attribute änderst
Heb den jeweils ersten Stichpunkt der beiden Listen hervor, indem du dessen Farbe oder ähnliches änderst
Verändere mithilfe von Pseudoklassen, z.B. die Hintergrundfarbe eines Input Feldes, wenn der Nutzer darüber „hovert“
Zuletzt kannst du den Hintergrund vom Footer ändern und den Unterstrich im Link entfernen
Flexbox
Vertiefe deine Flexbox Kenntnisse mit CSS Froggy (https://codepip.com/games/grid-garden/). Spiele alle 24 Level auf dieser Website, um das Erlernte in die Praxis umzusetzen.
Wenn du Tower Defense Spiele magst, kannst du auch noch in https://flexboxdefense.com reinschauen.
Wenn dir Flexbox Froggy gefallen hat, empfehlen wir dir auch noch mithilfe Grid Garden (https://codepip.com/games/grid-garden/) das CSS Grid System zu verstehen. Dieser Teil ist jedoch optional.
